###### 基于Unity3D的翻书效果
使用Unity中的2D特性模拟“3D翻书效果”。

## 功能和效果

- 支持左向右/右向左翻书。
- 支持自动翻书。
- 支持移动端横竖屏适配。
- 只支持单点触控。

具体见Demo演示。

## 关键技术

**Mask**——UGUI里的遮罩组件，用来使图片等只显示一部分；

EventTrigger——点击、拖拽等交互事件解决方案；

**DOTween**——第三方补间动画插件；

贝塞尔曲线原理——计算运动轨迹；

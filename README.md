# 基于Unity3D的翻书效果
使用Unity中的2D特性模拟“3D翻书效果”。

## 功能和效果

- 支持左向右/右向左翻书。
- 支持自动翻书。
- 支持移动端横竖屏适配。
- 只支持单点触控。

具体见Demo演示。

## 用到的技术

UGUI——GUI、遮罩、横竖屏适配解决方案；

EventTrigger——点击、拖拽等触控交互事件解决方案；

DOTween——补间动画解决方案（第三方插件）；

贝塞尔曲线原理——计算书页运动轨迹，辅助补间动画的实现；
